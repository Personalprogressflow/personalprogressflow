
# Personal Progress Flow 🎭🎵

**A Mood Detection & Recommendation Web Application**

---

## 📌 Project Overview

**Personal Progress Flow** is a mini project developed by a team of 3rd year AIML students (2024–2025) from Acharya Nagarjuna University. The application helps users recognize their emotional state from a selfie and receive personalized motivational suggestions and Telugu songs to reflect or uplift their mood.

---

## 🚀 Features

1. **Image Upload or Camera Capture**
   - User can upload a selfie or capture one using the browser interface.

2. **Mood Detection**
   - The image is processed by a lightweight ML model via a Flask backend.
   - It predicts one of 7 moods:
     - 😄 Happy
     - 😢 Sad
     - 😠 Angry
     - 😲 Surprise
     - 😨 Fear
     - 🤢 Disgust
     - 😐 Neutral

3. **Personalized Suggestions**
   - Based on detected mood, the frontend displays:
     - A motivational suggestion.
     - 5 embedded Telugu songs aligned with the mood.

**Example:**  
Student comes home stressed → Opens the app → Uploads a selfie → Mood detected as “Sad” → App suggests uplifting songs like *“Nee Kallu Neeli Samudram”* → Mood lifted 🎧🙂

---

## 🧠 Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python (Flask)
- **Machine Learning**: Lightweight emotion detection model (TensorFlow/Keras)
- **Hosting**:
  - Frontend: Firebase Hosting
  - Backend: Render
